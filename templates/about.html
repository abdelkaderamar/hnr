HNR (HackerNews Reader)

HNR is my personnal app to read Hackernews website

The requirements behind building this application were:

  • Save stories so I can read them later
  • Hide already visited or non intersting stories
  • Export saved stories in a textual format
  • Create list by keywords
  • Feed my link aggregator Kopikol

It is a self-hosted Django application. The source code can be found here
